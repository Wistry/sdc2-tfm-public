# Benchmark extendido

## Objetivo

Comparar modelos tabulares con los feature sets SoFiA, extendidos y exclusivamente espectrales.

## Entrada

- Dataset extendido limpio del catálogo permisivo.
- Etiquetas internas TP/FP limpias.

## Script

- `scripts/04_benchmark_extended_features.py`

## Salida

- `extended_benchmark_results.csv`
- `extended_feature_set_comparison.csv`

Las métricas son internas y no sustituyen el score oficial SDC2.
