# 01 - Comprensión del cubo FITS

## Objetivo

Comprobar la representación `cube[z, y, x]`, la WCS y la conversión entre
coordenadas físicas y de píxel.

## Entradas

- Cubo FITS de desarrollo.
- Truth catalogue de desarrollo.

## Contenido

- `notebooks/`: inspecciones reproducibles del cubo y una fuente truth.
- `figures/`: figuras exploratorias derivadas.
- `tables/`: tablas auxiliares derivadas.

## Notas de reproducibilidad

Los notebooks no forman parte del pipeline automático. Pueden contener outputs
embebidos y requieren configurar las rutas de datos antes de ejecutarse.
